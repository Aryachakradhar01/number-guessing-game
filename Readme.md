# Number Guessing Game with Python

A beginner-friendly command-line game built using Python.

## How It Works
- The program generates a random number between 1 and 99
- The user guesses until the correct number is found
- Hints are given for high or low guesses

## Tech Used
- Python

## What I Learned
- Python loops and conditionals
- Handling user input
- Writing simple game logic
